# mta-new-design

## Installation

This assumes that [node version manager](https://github.com/nvm-sh/nvm) has been installed already.

```
nvm install 16
nvm use 16
npm install
```

## Developing

```
npm run dev
```

Open browser to `http://localhost:3000/`. Browser should reload automatically when you update the source files.

## Deploying

```
npm run build
npm run deploy
```
